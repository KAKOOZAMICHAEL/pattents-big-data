# Dashboard Quick Start Guide

## ✓ Dashboard Ready to Use!

All required CSV files have been generated by the pipeline and are ready for the dashboard.

## Quick Start (3 steps)

### 1. Ensure Streamlit is Installed

```bash
pip install streamlit
```

### 2. Run the Dashboard

```bash
streamlit run dashboard.py
```

### 3. Open in Browser

Navigate to: `http://localhost:8501`

## Troubleshooting

### Issue: "StreamlitAPIException"

**Solution:** Ensure Python environment is activated

```bash
cd "c:\Users\miche\Downloads\Big Data"
python -m streamlit run dashboard.py
```

### Issue: "ModuleNotFoundError: streamlit"

**Solution:** Install streamlit

```bash
pip install streamlit
```

### Issue: File not found errors

**Solution:** Verify CSV files exist

```bash
Get-ChildItem *.csv
```

Should show 5 CSV files listed above.


2. Run pipeline:

3. Refresh dashboard (automatic rerun or press R)
